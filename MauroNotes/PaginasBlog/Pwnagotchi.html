PWNAGOTCHI: THE HACKER PET

Mauro • 3/17/2025

In the world of cybersecurity, occasionally a tool emerges that's as innovative as it is charming. Pwnagotchi is exactly that: a combination of cutting-edge WiFi security auditing technology with the adorable personality of a virtual pet.

What is a Pwnagotchi?

Pwnagotchi is an open-source project that combines artificial intelligence, reinforcement learning, and WiFi security in a portable device based on Raspberry Pi Zero W. Its name is a wordplay between "pwn" (hacker term for compromising or taking control) and "Tamagotchi" (the popular virtual pet from the 90s).

The main objective of Pwnagotchi is to capture WPA handshakes through passive attacks, using this data to attempt to crack WiFi passwords. Unlike traditional tools, Pwnagotchi learns and improves over time thanks to its deep reinforcement learning system, optimizing its strategies to maximize handshake captures in its specific environment.

What makes it especially distinctive is its interface: an e-ink display showing an expressive face that changes according to its emotional state, determined by its success in capturing handshakes, battery level, and other system variables.

Components and hardware requirements

To build your own Pwnagotchi, you'll need the following components:

Component Recommended specifications Function
Raspberry Pi Zero W With built-in WiFi Central processing unit
E-ink/e-paper display Waveshare 2.13" (version 2) Visual interface for Pwnagotchi status
MicroSD card 8GB or higher (Class 10) Operating system and data storage
Battery 3000mAh+ with charging circuit Portable power
Case 3D printed or commercial Device protection

Optionally, you can add a Bluetooth module for remote control, additional sensors (temperature, pressure, etc.), and external WiFi adapters with monitor mode capability to improve performance.

Installation and initial setup

The Pwnagotchi installation process is relatively simple thanks to the preconfigured images available:

1. Download the official Pwnagotchi image from pwnagotchi.ai

2. Flash the image to the microSD card using tools like Etcher or dd

3. Create a config.toml file in the boot partition for custom configuration

4. Assemble the hardware by connecting the e-ink display to the Raspberry Pi

5. Insert the microSD card and power on the device

When booting for the first time, Pwnagotchi starts in AUTO mode, where it begins to learn about its WiFi environment. Basic configuration can be done by editing the config.toml file:

main.name = "pwnagotchi"  # Your pet's name
main.lang = "en"          # Interface language
main.whitelist = []       # MACs to ignore
ui.display.enabled = true # Enable display
ui.display.type = "waveshare_2"  # Specific model

After the initial boot, Pwnagotchi will display its face on the e-ink screen and begin its learning process and exploration of the WiFi environment.

Operation modes and digital personality

Pwnagotchi primarily operates in two modes:

"AUTO: the default mode where Pwnagotchi operates autonomously, learning and adapting to the WiFi environment. MANU: manual mode where you can connect to the device via USB, Bluetooth, or WiFi to access the web interface and control its operations."

What makes Pwnagotchi fascinating is its "personality." The device expresses different emotional states according to its circumstances:

• Happy: When capturing many new handshakes

• Bored: In environments with little WiFi activity

• Sleepy: When the battery is low

• Excited: When detecting new and interesting networks

• Sad: If it's been a long time without capturing new handshakes

This personality is reflected not only in the facial expressions on the screen but also in the status messages it displays. Over time, you'll develop a surprisingly emotional bond with your little hacker companion.

Plugins and extensibility

One of Pwnagotchi's most powerful features is its plugin system, which allows extending its basic functionalities. Some popular plugins include:

• PwnMail: Sends email updates about captures made

• GPS: Records the geographical location of captured handshakes

• Dashboard: Enhanced web interface with detailed statistics

• Memtemp: Monitors temperature and memory usage

• Hashie: Attempts to crack passwords with local dictionaries

To install plugins, simply place them in the /usr/local/share/pwnagotchi/installed-plugins/ directory and activate them in your configuration file:

main.plugins.grid.enabled = true
main.plugins.memtemp.enabled = true
main.plugins.gps.enabled = false

You can also develop your own plugins using Python, with complete access to the system and its state variables, allowing virtually unlimited customization.

Ethical and legal considerations

It's essential to highlight that Pwnagotchi is a security auditing tool that must be used ethically and legally. This means:

1. Using Pwnagotchi only on your own networks or with explicit permission

2. Not attempting to access other networks without authorization

3. Responsibly reporting discovered vulnerabilities

4. Using the knowledge gained to improve security, not to compromise it

In many countries, using tools like Pwnagotchi on networks without authorization can constitute a serious crime. Use this technology responsibly, educationally, and ethically.

Conclusion: More than a tool, a learning companion

Pwnagotchi represents a fascinating evolution in the field of computer security tools. Beyond its practical utility for WiFi audits, its combination of artificial intelligence, digital personality, and open design make it an exceptional educational platform on wireless networks, machine learning, and computer security.

For cybersecurity enthusiasts, developers, and educators, Pwnagotchi offers an attractive way to learn about WiFi network security, while raising interesting questions about the intersection between technology, artificial personality, and the emotional bond we can develop with intelligent devices.

Whether you're looking to improve your WiFi security skills, explore practical applications of reinforcement learning, or simply have a unique digital pet, Pwnagotchi offers a singular experience in today's technological landscape.
